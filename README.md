# ByteForce - MediQueue
South African Community Clinic Appointment and Queue Management System (SD group project).

## Features
- Login page
- Registration page (First Name, Surname, ID, DOB validated vs ID, Password, Details confirmation)

## Run
Go to the backend folder in the terminal, run 'node server.js', go to a browser and open http://localhost:3000/ and then the name of the html file

## Users
Registrations saved in browser localStorage (clears on clear data).
